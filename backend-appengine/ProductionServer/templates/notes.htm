Statistics API


Get all Userdata
	POST Data - 'passphrase=77777777' to URL: /receive/serverstatistics
	curl example: curl --data 'passphrase=43870193' http://localhost:8082/receive/serverstatistics
	curl example: curl --data 'passphrase=43870193' https://itsgoingupdev.appspot.com/receive/serverstatistics
	Notes: 
	Output Syntax: Devices  Stocks  NotificationsSet  TotalNotificationssent

App Interface API


Register New Device
	POST Data - 'deviceToken=1ac16ea496f514e935e93daac049a464ef041705d30a4d809338786cb2be1814' to URL: /data/receive
	curl example: curl -F 'deviceToken=1ac16ea496f514e935e93daac049a464ef041705d30a4d809338786cb2be1814' https://itsgoingup-dev.appspot.com/receive/registerdevice
Set/Change Notification
	POST Data - 'deviceToken=1ac16ea496f514e935e93daac049a464ef041705d30a4d809338786cb2be1814' to URL: /data/receive
	curl example: curl --data 'notificationID=E2AAFB78FDCA&deviceToken=1ac16ea496f514e935e93daac049a464ef041705d30a4d809338786cb2be1814&ticker=BHP.AX&setPoint=39.000&aboveOrBelow=A&repeat=No' http://localhost:8082/receive/setnotification    

	Notes: 
	setPoint is value of the stock. Should be 3 decimal places
	aboveOrBelow = A/B
	repeat = No/Hr/D/W
	notificationID must be unique - 12 digits
Delete Single Notification
	POST Data - 'notificationID=4DELETION000&deviceToken=1ac16ea496f514e935e93daac049a464ef041705d30a4d809338786cb2be1814&delete=1' to /receive/setnotification
	curl example: curl --data 'notificationID=4DELETION000&deviceToken=1ac16ea496f514e935e93daac049a464ef041705d30a4d809338786cb2be1814&delete=1' http://localhost:8082/receive/setnotification
Note: 4DELETION is the notificationID
